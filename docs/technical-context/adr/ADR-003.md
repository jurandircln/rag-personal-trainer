# ADR-003: sentence-transformers para Geração de Embeddings

**Data:** 2026-03-15
**Status:** Aceito
**Autores:** Time Jarvis

---

## Contexto

O projeto Jarvis precisa gerar embeddings para:
- Indexar chunks de documentos em português (textos de treino, anamneses, metodologias)
- Codificar queries dos usuários para busca semântica
- Operar sem custo por chamada de API

---

## Decisão

Adotar **`paraphrase-multilingual-mpnet-base-v2`** via biblioteca `sentence-transformers` para geração de embeddings.

- Dimensão dos vetores: **768**
- Execução: local (CPU ou GPU)
- Configuração: variável `EMBEDDING_MODEL` no `.env` (permite troca sem alterar código)

---

## Consequências

### Positivas
- **Suporte a português:** modelo multilingual treinado em 50+ idiomas incluindo pt-BR
- **Sem custo:** roda localmente, sem cobrança por token
- **Qualidade adequada:** bom desempenho em similaridade semântica para domínio de fitness
- **Configurável:** modelo pode ser substituído via variável de ambiente

### Negativas / Riscos
- **Download inicial:** ~420 MB no primeiro uso
- **Latência na indexação:** mais lento que embeddings via API para grandes volumes
- **Dimensão 768:** maior que text-embedding-ada-002 (1536 dims), mas suficiente para o volume de dados do projeto

---

## Alternativas Descartadas

| Opção | Motivo da Rejeição |
|---|---|
| OpenAI text-embedding-ada-002 | Custo por token, dependência de API externa |
| Cohere Embed | Custo por chamada |
| Modelos somente em inglês | Qualidade inferior para textos em português |
| NVIDIA NIM embeddings | Adiciona dependência à mesma infraestrutura do LLM |
