# Glossário — Jarvis Personal Trainer

## Termos do Domínio

| Termo | Definição |
|---|---|
| Anamnese | Ficha de avaliação inicial do aluno com histórico de saúde, objetivos e restrições |
| Chunk | Fragmento de texto extraído de um documento para indexação vetorial |
| Embedding | Representação vetorial de um texto que captura seu significado semântico |
| RAG | Retrieval-Augmented Generation — técnica que combina busca em base de conhecimento com geração de texto por LLM |
| Vector Store | Banco de dados especializado em busca por similaridade vetorial (usamos Qdrant) |
| Personal Trainer | Profissional de educação física especializado em treinamento individual |

## Termos Técnicos

| Termo | Definição |
|---|---|
| LLM | Large Language Model — modelo de linguagem de grande escala |
| NVIDIA NIM | Plataforma de inferência de modelos de IA da NVIDIA com API OpenAI-compatible |
| Qdrant | Vector database open-source usado como backend de busca semântica |
| LangChain | Framework Python para orquestração de pipelines com LLMs |
| Streamlit | Framework Python para criação rápida de interfaces web |
